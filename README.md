# Restaurant-Review-Sentiment-Analysis
End-to-end NLP pipeline for Restaurant Review Sentiment Analysis 🍴🤖. Includes preprocessing, Bag-of-Words &amp; TF-IDF vectorization, multiple classification algorithms (Logistic Regression, Naive Bayes, SVM, KNN, Random Forest, Decision Tree, XGBoost, LightGBM), and model evaluation with cross-validatio.
